📰 Daily Sentinel
Um portal de notícias fictício inspirado no universo dos quadrinhos e da cultura pop (Marvel), focado em reportagens sobre os Vingadores, Wakanda e o Multiverso. O projeto foi desenvolvido para praticar conceitos de layout moderno na web.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica da página (<header>, <main>, <article>, <aside>, <footer_>).

CSS3: Estilização global, efeitos de hover e posicionamento.

CSS Grid Layout: Utilizado para a malha principal do site (grid-template-areas) e para a organização dos cards de notícias.

Flexbox: Aplicado no cabeçalho e nos componentes internos para o alinhamento de elementos.

🛠️ Funcionalidades e Estrutura
Destaque Principal: Notícia central com imagem, resumo e metadados (autor e data).

Grid de Notícias: Seção de cartões que se ajustam para exibir matérias secundárias.

Barra Lateral (Aside): Ranking das notícias "Mais lidas" do portal.

Responsividade Base: Uso de Grid e Flexbox para facilitar a adaptação do layout.

📂 Como Executar o Projeto
Baixe ou clone este repositório.

Certifique-se de ter a pasta assets com as imagens necessárias (logo.png, logo2.png, avengers.png).

Abra o arquivo index.html diretamente em qualquer navegador de sua preferência.
